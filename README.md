# Hava Durumu - Turkce Hava Durumu Platformu

Profesyonel, modern ve tamamen Turkce bir hava durumu web uygulamasi.

## Ozellikler

- **Sehir Arama**: Turkiye ve dunya genelinde sehir arama
- **Anlik Hava Durumu**: Gercek zamanli sicaklik, nem, ruzgar ve hava kosullari
- **5 Gunluk Tahmin**: Gunluk maksimum/minimum sicaklik ve hava durumu ikonlari
- **Saatlik Grafik**: 24 saatlik sicaklik degisimi grafigi (Recharts)
- **Detayli Meteoroloji Verileri**: Nem, ruzgar hizi, basinc, UV indeksi, gorus mesafesi, yagis
- **Konum Algilama**: Tarayici geolocation ile otomatik konum tespiti
- **Gunduz/Gece Temasi**: Saate gore dinamik arka plan gradyanlari
- **Responsive Tasarim**: Mobil ve masaustu uyumlu

## Teknoloji

- **Frontend**: React 19, TypeScript, Vite, Tailwind CSS v4
- **Grafikler**: Recharts
- **API**: Open-Meteo (ucretsiz, API anahtari gerektirmez)
- **Geocoding**: Nominatim OpenStreetMap (ters geocoding)
- **UI Bilesenleri**: shadcn/ui (Radix UI)
